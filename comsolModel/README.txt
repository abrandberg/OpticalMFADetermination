Four models for calculating the polarization state of light as it passes through various
idealizations of the pulp fiber wall.

linearOptics_v4_multiLayer.mph
	Contains three layers (S1, S2, S3).

linearOptics_v2.mph
	"Standard" model with only S2.

linearOptics_waveplate.mph
	Special model for the case of waveplates.

linearOpticsTutorial.mph
	Legacy model kept for reference.
